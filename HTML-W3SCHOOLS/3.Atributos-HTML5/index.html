ATRIBUTOS EN HTML

Los atributos provee de informacion adicional acerta de los elementos HTML

Entre los atriibutos tenemos

  • Todos los elementos de HTML puden tener atributos
  • Los atributos proveen de informacion adicional acerca de un elemento
  • Los atributos son siempre especificados al empezar un tag
  • Los atributos generalmente vienen en pares de nombres /valor como: nombre="Valor"

EL Atributo href

En HTML los links son definidos con la etiqueta (a). El link es una direccion especifica en el atributo href: Por ejemplo.

Ir a w3schools

El Atributo src

Html define las imagenes con la etiqueta (img). El nombre del archivo fuente de la imagen es especificado en el atributo src.

Atributo Style

El atributo style es usado para especificar los estilos de un elemento, preferencia color, fuente de palabra y tamaño.

Yo soy un texto color rojo, utilizando el atributo Style.

Atributo Lang

El lenguaje de un documento puede ser declarado in la etiqueta HTML. El lenguaje es declarado con el atributo Lang. La declaracion del lenguaje es nuy importante para acceder a aplicaciones y motores de busqueda.

Atributo Title

El atributo title es agregado en el elemento P. El valor del atributo title serà mostrado como un elemento flotante cuando el mouse sobrepase el parrafo.
